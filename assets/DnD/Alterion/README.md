# Alterion

## Use the interactive Map

Go to [Azgaar's Fantasy Map Generator](https://azgaar.github.io/Fantasy-Map-Generator/).
Open the menu and at the bottom select "Load". Choose to load from a URL and paste the
URL [Alterion.map](./Alterion/Alterion.map) in the text box to load the official map.
